# **App Name**: Сумрак Истории

## Core Features:

- Навигация по зонам: Плавное перемещение между взаимосвязанными зонами для стимулирования исследования.
- Раскрытие сюжета: Записки или призрачные встречи для передачи ключевых элементов повествования пользователю. В одной из записок будет спрятано поздравление с днем рождения.
- Атмосферное погружение: Гнетущая, напряженная, жуткая атмосфера с ограниченной видимостью и дезориентирующими окружающими звуками.
- Персонализированное поздравление: Инструмент LLM, который определяет уникальное поздравительное сообщение в зависимости от того, как много и как хорошо игрок исследовал мир. Это будет особенное поздравление с днем рождения, которое можно будет редактировать через код.

## Style Guidelines:

- Приглушенный синий (#4683ab) для создания мрачной и таинственной атмосферы.
- Темно-серый (#222222) для создания темной и гнетущей атмосферы.
- Нежный ржавый (#8d4f15) для интерактивных элементов и точек интереса.
- 'Literata', переходный шрифт с засечками, создающий винтажное и литературное ощущение.
- Используйте минималистичные значки для обозначения ключевых предметов или навигационных точек.
- Тонкие, жуткие анимации, такие как качающиеся предметы или призрачные явления, для усиления атмосферы.
